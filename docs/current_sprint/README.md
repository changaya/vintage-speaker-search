# Current Sprint

이 폴더는 현재 진행 중인 Sprint의 작업 문서를 저장합니다.

---

## 파일 형식

### ORDER{MMDD}.md
새 기능 개발 문서

**예시**: `ORDER121701.md` (12월 17일 첫 번째 작업)

### REFACTORING{MMDD}.md
리팩토링 문서

**예시**: `REFACTORING121701.md` (12월 17일 첫 번째 작업)

---

## Sprint 종료 시

1. 작업 문서 검토 및 "작업 결과" 섹션 완성
2. `docs/sprints/SPRINT_{YYYY_MM_DD}.md` 요약 생성
3. 파일 이동:
   - `docs/sprints/sprint{N}/` (권장)
   - 또는 `backup/` 폴더
